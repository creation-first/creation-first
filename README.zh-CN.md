# 先创主义 Creation-First

> AI 把「造」的成本降到零。人只需要「创」。
> 技能不再是创造的前提，而是创造的副产品。这就是先创主义。

---

## 什么是先创主义

在过去的教育范式中，人必须先学会技能，才被允许去创造。画画要先学素描，编程要先学语法，写作要先学修辞。

这套「先学后创」的逻辑统治了人类教育数千年。

**AI 改变了这一切。**

当一个 10 岁的孩子可以用自然语言指挥 AI 生成一款游戏、一首歌、一部动画，「技能门槛」就不再是创造的入口。

**意图成为唯一的起点。创造成为学习的方式。技能在创造的过程中自然习得。**

这不是对学习的否定，而是对学习顺序的颠覆 —

**先创后学。造即最深度的学。**

---

## 核心主张

🔨 **创造先于技能** — 不会不是不做的理由，AI 让任何人都能从意图直达作品

🌱 **学习是创造的副产品** — 在造的过程中理解原理，比在课堂上背诵公式深刻一百倍

🔄 **范式已经转移** — 从「先学后创」到「先创后学」，不是选择，是 AI 时代的必然

🔓 **思想应该开源** — 先创主义不属于任何人，任何人都可以 fork、演绎、实践

---

## 先学说 vs 先创说

| | 先学说（旧范式） | 先创说（新范式） |
|---|---|---|
| **起点** | 技能 | 意图 |
| **路径** | 学习 → 掌握 → 创造 | 创造 → 遇到问题 → 学习 |
| **门槛** | 高（需要长期训练） | 零（AI 辅助即可起步） |
| **动力** | 外驱（考试、证书） | 内驱（好奇心、表达欲） |
| **技能** | 前提条件 | 自然副产品 |
